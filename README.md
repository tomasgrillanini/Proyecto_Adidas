# 👟 Análisis de Performance Retail: Adidas

## 📝 Descripción del Proyecto
Este es mi primer proyecto y consiste en un análisis integral de ventas y rentabilidad de **Adidas**. El objetivo principal fue transformar datos de ventas dispersos en una herramienta de inteligencia de negocios que permitiera identificar los métodos de venta más efectivos, las regiones con mayor margen y el cumplimiento de objetivos mediante indicadores visuales.

## 🛠️ Tecnologías y Herramientas
* **Power BI Desktop**: Construcción del modelo de datos y visualizaciones.
* **DAX (Data Analysis Expressions)**: Creación de medidas para calcular ingresos totales, beneficios operativos y tasas de eficiencia.
* **Power Query**: Limpieza de datos y estandarización de categorías de productos y regiones.
* **Custom Visuals & UX**: Implementación de *Chiclet Slicer*, *Tachometer* y un menú de navegación optimizado para mejorar la experiencia del usuario entre las distintas secciones del informe.

## 📊 Hallazgos y KPIs Clave
* **Análisis General**: Visualización macro de los ingresos y beneficios operativos con navegación corregida.
* **Eficiencia Operativa**: Análisis detallado por categoría de producto (Calzado, Ropa, Accesorios).
* **Segmentación Geográfica**: Identificación de las ciudades y regiones con mejor rendimiento.
* **Métodos de Venta**: Comparativa de rendimiento entre ventas Online, In-store y Outlet.

## 🎨 Diseño y UX (Experiencia de Usuario)
El reporte cuenta con una interfaz personalizada que respeta la identidad de marca de Adidas. Se incluyó un **menú lateral de navegación** que permite al usuario desplazarse de manera fluida entre el Análisis General, el Comportamiento Comercial y las Conclusiones.
